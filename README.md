# Határidõnapló (Deadline Diary)

A Python application for managing events and deadlines.

## Features
- Create, modify and delete events
- Store event date, time, location, name and notes
- List events chronologically
- Filter events by day, week or month
- Search events by name

## Installation
1. Clone the repository
2. Install requirements: pip install -r requirements.txt
3. Run the application: python src/main.py

## Project Structure
- src/ - Source code
- tests/ - Test files
- data/ - Event storage
